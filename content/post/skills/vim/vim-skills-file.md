## 编辑文件过程

文件从磁盘调读取用到内存，创建缓冲区文件映像, 修改作用在缓冲区中, 通过保存命令将修改作用到磁盘

## 命令

### `:ls` 显示缓冲区列表

   ```
   1      "setup.sh"                     line 4
   2 #    "test.txt"                     line 3
   3 %a + "vim-skills-01.md"   
   ```
   % 表示当前显示的文件 # 表示轮换文件 + 表示缓冲区内容为保存

### 切换文件命令

   * `bn, bnext` 下一个
   * `bp, bprev` 上一个
   * `bf, bfirst` 第一个
   * `bl, blas` 最后一个
   * `b n` 切换到缓冲区列表中序列为 n 的文件
   * `<c-^>` 切换到轮换文件
   * `b 正则` 打开路径符合正则的文件

   b 指的是 buffer  所有命令可通过 `<tab>` 补全

### 删除缓冲区文件

   * `bd, bdelete` 

      示例 `bd n1 n2 n3` 和 `n1,n2 bd`

### 隐藏缓冲区

   * `!` 会隐藏未保存的缓冲区

   * `wall!`  写入所有未保存文件

   * `qall!` 不保存写入文件退出

   * `edit!` 重新从磁盘读取该文件内容，覆盖原缓冲区

   * `set hid` 设置切换到下一个缓冲区时默认隐藏当前缓冲区 `:h hid` 查看帮助

### 参数列表管理文件

   * `:argdo` 对参数列表中所有文件执行一条 Ex 命令
   * `:args` 显示参数列表 `[]` 扩起来的表示当前活动文件  
   * `:args {arglist}` 清空原参数列表, 添加指定文件到参数列表和缓冲区中
      
	  `:args \`ls\`` 添加当前目录下所有文件
   * `:next, :prev` 遍历参数列表(prev 不可缩写)
	  
### 通配符

   * `*` 匹配任意个字符， 不递归进入子目录
   * `**` 匹配任意个字符, 递归进入子目录
