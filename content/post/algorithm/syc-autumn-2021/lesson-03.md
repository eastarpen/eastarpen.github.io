## 单调栈

维护一个栈，栈内元素是单调的

加入一个新元素时, 先(自栈顶)弹出所有大于等于v 的元素

```c++
while(Top > 0 && Stack[Top] >= Val) --Top;
Stack[++Top] = Val;
```

单调栈不考虑出栈

### 例1

给出数列 $a_n$;

对每个 $i$ 求最大的 $j$ 使得 $j < i$ 且 $a_j > a_i$

维护一个单调递减栈, 改栈即为可能的 $a_j$

## 单调队列

维护一个队列，队列内元素为单调的

加入一个新元素时， 自队尾弹出所有大于等于v的元素

```c++
while(Tail>Head && Queue[Tail-1] >= Val) --Tail;
Queue[Tail++] = Val;
```
