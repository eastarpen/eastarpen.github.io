As stated in gohugo official site

Hugo is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.

For those who knows little about how to build and maintain a web site or those who care nothing but write, hugo is your friend. With hugo, even a normal internet user can build his site. Besides, if you has some programming experience(like command-line and github usage experience), you can build and maintain your site easily and conveniently.

Preparation before we start

install hugo

hugo’s official guide tells you how to install hugo at all platforms.

But the most easy way is download releases at hugo github repository .

Tips

  1. remember add hugo binary file to your environment
  2. extended versions are more recommended
  3. packer managers(like apt in ubuntu) alwarys download old versions

install go and git

go and git are hugo Prerequisite Tools. You can easily download them at any platforms.

try google “install git at ubuntu” or “install go at windows”

build site locally

Hugo using command hugo new site myBlog to create local site directory. You can change “myBlog” to any name you like.

After run this command, a directory named “myBlog” should occur in your current directory. That is your local site directory.
